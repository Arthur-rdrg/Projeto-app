# Projeto-app

App mobile em React Native para conferencia de compras por imagem.

## Estrutura

- `frontend`: aplicativo mobile com Expo/React Native.
- `backend`: reservado para anotacoes. O projeto usa Firebase Authentication e Cloud Firestore; as imagens sao salvas localmente no celular.

## Primeiro passo

Entre na pasta do app:

```bash
cd frontend
```

Instale as dependencias, se necessario:

```bash
npm install
```

No PowerShell do Windows, se aparecer erro de script bloqueado com `npm.ps1`, use:

```bash
npm.cmd install
```

Rode o projeto:

```bash
npm start
```

Ou, no mesmo caso do PowerShell:

```bash
npm.cmd start
```

Depois leia o QR Code com o app Expo Go no celular.
